apk反编译

标签（空格分隔）： Android

###工具
apktool
dex2jar
jdgui

###apktool（Get XML files from APK）

  • Mac OS X:
  1. Download Mac wrapper script (下载脚本，并保存，文件名为apktool.sh)
  2. Download apktool-2 (find newest here)
  3. Rename downloaded jar to apktool.jar(下载的jar包重命名apktool.jar)
  4. Move both files (apktool.jar & apktool) to /usr/local/bin (root needed)(移动apktool.jar和apktool.sh到/usr/local/bin方便全局使用，我采用alias配置一下alias apktool=apktool.sh的绝对路径，当前terminal可用)
  5. Make sure both files are executable (chmod +x)(对apktool.sh添加可执行的权限chmod +x apktool.sh)
  6. Try running apktool via cli(运行apktool ok)
  7. apktool d apkname.apk(得到反编译的xml文件)

dex2jar(Get Java files from APK)

  1. Rename the .apk file into .zip file (example SharedPr.apk into SharedPr.zip)(重命名后缀名.apk改为.zip).
  2. Extract SharedPr.zip file and copy classes.dex file from extracted folder.(解压改名后的zip文件，复制classes.dex)
  3. Extract dex2jar.zip and paste classes.dex into dex2jar folder. (把复制的classes.dex文件paste 到dex2jar文件夹下)
  4. Open command prompt and change directory to the dex2jar folder. Then write dex2jar classes.dex and press enter. Now you will get classes-dex2jar.jar file in the same folder.(打开terminal，移动到dex2jar文件夹下，执行dex2jar classes.dex，回车，得到classes-dex2jar.jar文件)

jdgui(打开dex2jar得到的jar文件)

  1. Now double click on jd-gui(Java decompiler) and click on open file. Then open classes-dex2jar.jar file from that folder(打开jdguid软件，打开文件选中classes-dex2jar.jar,可以看到java .class文件了)
